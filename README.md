# VersionedMap
Multi-version document store. Fundamental tool to build varials anonymous online services.

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'versioned_map', git: 'https://github.com/mindpin/versioned_map.git'
```

And then execute:

    $ bundle
